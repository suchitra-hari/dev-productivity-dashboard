Methodology
Tool Adoption % = engineers active on Cursor or Claude Code in the 30-day window ÷ DX-tracked team size. "Active" = is_active = true on at least one day in the window (cursor_daily_user_metrics / claude_code_daily_user_metrics).

Agentic PR % = PRs where the author had an active Cursor or Claude session on the same calendar day the PR was merged. This is a conservative proxy — it undercounts PRs where AI was used earlier in the branch lifetime but not on merge day.

Tier thresholds: <30% = Exploring  |  30–59% = Adopting  |  ≥60% = Full-Agentic. "Primary Tool" shows which tool had more unique active users per team (Claude / Cursor / Both if equal).

Note on Leadership & AI Foundations: shown for completeness — non-pillar groupings with mixed membership. High adoption reflects Claude licensing breadth, not necessarily day-to-day coding tool usage.
